项目架构 ​

系统架构图 ​

本项目采用微服务架构,主要包含以下服务:

  • 用户服务(User Service)
  • 订单服务(Order Service)
  • 支付服务(Payment Service)
  • 景点服务(Attraction Service)
  • 评论服务(Comment Service)

技术架构 ​

  • 后端: SpringBoot3 + Spring Cloud Alibaba
  • 前端: Vue3 + ElementUI Plus
  • 数据库: MySQL + Redis
  • 消息队列: RocketMQ
  • 注册中心: Nacos
